### **第一部分：核心准则**
在GLP报告的仪器与设备章节中，必须列出所有用于试验的关键仪器、设备型号及设备编号，确保信息来源于验证方案或原始记录。静态文本（如设备名称）直接复制自方案；动态文本（如具体编号）从附件或实验室记录中提取。若无方案细节或偏差，声明“无偏离”并仅报告已确认设备。严禁省略任何已验证的设备项，确保完整性与可追溯性。

### **第二部分：通用模板**
从验证方案或原始记录中提取关键仪器与设备列表，按类别组织撰写报告段落。使用以下指令脚本执行：

1. 扫描附件[设备清单]或方案章节，识别主要仪器类别（如质谱仪、液相色谱系统、分析天平、色谱柱）。
2. 对于每个类别，提取设备名称、型号及设备编号{{Equipment_ID}}，并按报告格式排列。
3. 若无偏差，添加声明“无偏离”；若有偏差，标注{{Deviation_Description}}并说明影响。
4. 填入通用模板：

{{Instrument_System}}（设备编号：{{System_ID}}）  
分析天平：[{{Balance_Brand}}, {{Balance_Model}}（设备编号：{{Balance_ID}}）]  
色谱柱：{{Column_Brand}} {{Column_Type}} ({{Dimensions}}, {{Particle_Size}})
