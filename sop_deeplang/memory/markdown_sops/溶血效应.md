### **第一部分：核心准则**
溶血效应章节的撰写必须基于验证方案的测定结果，使用客观描述性语言报告检测数据和符合性判定。接受标准由验证方案预定义，报告中仅陈述结果是否符合，无需列出具体数值阈值。若无偏差或异常，直接声明符合性；若存在不符合情况，需注明具体偏差并追溯原因。数据来源限于表23等指定附件，确保描述简洁、事实导向，不添加未验证的解释。

### **第二部分：通用模板**
1. 从验证方案中提取溶血效应的测定方法和接受标准描述。
2. 从原始数据附件（如表23）中提取实际测定结果，包括浓度水平（如{{Concentration_Level}}）。
3. 比较实际结果与接受标准，生成符合性判定。
4. 填入以下模板语句：

“溶血效应按验证方案测定，结果见表{{Table_Number}}。结果表明，{{Concentration_Level}}溶血符合接受标准。”

分支逻辑：若不符合，替换为“结果表明，{{Concentration_Level}}溶血不符合接受标准，偏差为{{Deviation_Description}}，原因分析见{{Section_Reference}}。”
