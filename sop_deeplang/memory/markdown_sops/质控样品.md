### **第一部分：核心准则**
* 基于验证方案中定义的分析批接受标准，报告质控样品（QCs）的测定结果。
* 仅描述符合标准的质控样品结果；若存在不符合情况，需明确列出不符合项并说明原因。
* 结果表述必须客观、基于数据，不引入额外解释；所有批次均需覆盖，无遗漏。

### **第二部分：通用模板**
从验证方案中提取质控样品测定结果，填入以下模板：

“除准确度及精密度分析批的质控样品按验证方案测定，除准确度及精密度分析批的质控样品结果见表{{Table_Number}}。结果表明，所有{{Compliant_Batches}}的质控样品均符合分析批接受标准。”

* 数据流转：从原始记录提取质控样品数据 -> 确认批次合规性 -> 插入{{Table_Number}}（指向结果表格）及{{Compliant_Batches}}（如“接受批”） -> 若不符合，添加分支：“{{Non_Compliant_Batches}}的质控样品不符合标准，详见表{{Table_Number}}。”

### **第三部分：报告示例**
除准确度及精密度分析批的质控样品按验证方案测定，除准确度及精密度分析批的质控样品结果见表10。结果表明，所有接受批的质控样品均符合分析批接受标准。
