### **第一部分：核心准则**
* 严格遵守FDA 21 CFR Part 11/GLP法规，确保报告中数据完整性、准确性和可追溯性，所有数据来源于原始记录（Raw Data），禁止手动篡改。
* 区分静态文本（直接引用协议接受标准）和动态数据（从实验记录提取实际结果）。
* 判定逻辑：计算实际结果与协议接受标准的符合性，若符合则使用“Meets Acceptance Criteria”表述；若不符合，明确报告“Does Not Meet”并追溯原因。
* 数据修约：百分比保留1位小数，日期格式为“Month DD, YYYY”，均值（Mean）和变异系数（CV）保留适当有效位（Mean至小数点后1-2位，CV至1位小数）。
* 分支处理：包含“Conforms”和“Non-conforms”双分支，确保SOP逻辑闭环覆盖所有可能场景。
* 引用规范：协议用“Protocol §X.Y”，数据来源标注为“Attachment Z, Table W”。
