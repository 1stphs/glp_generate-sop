### **第一部分：核心准则**
* **定义**：该章节撰写时必须死守的"法律"。

材料和方法章节的撰写需严格遵循验证方案（VP）中描述的实验设计和操作细节，确保报告（VR）中呈现的信息完整、准确且与原始记录一致。核心要求包括：（1）直接引用VP中标准品、内标、仪器、溶剂/试剂、样品处理步骤等静态描述，不得修改或省略；（2）若无实际实验数据（如本例报告为空），则以声明形式概述方法概述和配制/处理流程，注明任何潜在更新将记录在原始记录中；（3）对于动态元素（如批号更新或新批次使用），必须从原始记录提取并标注，但通用化表述中避免具体数值，使用占位符如{{Batch_Number}}；（4）若报告无数据，仅确认方法符合指导原则（FDA 2018、EMA 2012、ICH M10 2022），并声明无偏离；（5）处理低信息量情况时，采用声明式表述，确保逻辑闭环：方法概述 → 关键组件描述 → 步骤序列 → 变更记录声明。严禁添加未在VP中出现的额外内容，如未指定的安全或归还细节仅在VP明确时纳入。
