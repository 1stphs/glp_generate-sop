### 一、核心填写规则
1. **内容定位**：该章节用于描述标准曲线样品和质量控制样品的配制情况，包括配制日期和储存条件。必须基于分析方法附录（附录II）进行引用，并在表格中列出所有历次配制记录。
2. **数据提取**：从原始记录或实验日志中提取配制类型（标准曲线样品或质量控制样品）、配制日期和储存条件。确保表格覆盖所有批次，无遗漏。
3. **合规性要求**：所有样品必须在已建立的稳定期内使用，并在表格后添加注记说明。储存条件通常为“现配现用”，如有偏差需额外说明（本章节无偏差情况）。
4. **通用化处理**：使用占位符表示日期和条件，例如{{Preparation_Date}}和{{Storage_Condition}}。表格格式固定为Markdown，确保列对齐。
5. **分支逻辑**：若无配制记录或所有样品现配现用且稳定，直接生成声明式描述；若有不符合稳定期的情况，需在注记中突出并评估影响。
