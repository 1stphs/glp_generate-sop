### **第一部分：核心准则**
在章节撰写中，必须确认整个实验过程符合预定方案，无任何异常或偏离发生。若存在任何未记录的特殊情况，则需补充描述并评估影响；否则，直接声明无特殊情况，以确保报告的完整性和合规性。准则适用于无数据输入场景，强调简洁声明而非详细分析。

### **第二部分：通用模板**
执行以下指令生成报告片段：  
1. 检查验证方案（Protocol）和实验记录，确认无任何异常、偏离或特殊事件发生。  
2. 若确认无特殊情况，使用以下模板语句填入占位符（如 [试验编号]）：  
   "本实验过程中无特殊情况发生。"  
3. 若存在特殊情况，扩展为："本实验过程中 [描述特殊情况]，已 [处理措施]。"  
4. 确保语句置于章节起始位置，采用正式商务语气，无需额外数据引用。

### **第三部分：报告示例**
本实验过程中无特殊情况发生。
