### **第一部分：核心准则**
* **定义**：该章节撰写时必须死守的"法律"。  
本章节用于报告内标对待测物的干扰测试结果，基于验证方案（VP）中的接受标准进行评估。核心合规底线：从原始数据（如Excel表5）提取干扰值，计算干扰率（干扰率 = (内标峰面积变化 / 基准峰面积) × 100%，修约至小数点后一位），并与VP中定义的接受标准（如±{{Acceptable_Interference_Limit}}%，例如±20%）对比。若所有结果均在标准范围内，判定为“符合”；若有偏差，需列出具体不符合项并说明原因。无数据时，直接声明“按验证方案测定，无干扰测试执行或结果符合标准”。必须包含VP中定义的所有数据折算因子（如游离型/盐型折算：游离型含量 = 盐型含量 × {{Salt_to_Free_Factor}}，其中{{Salt_to_Free_Factor}}为分子量比，例如盐型分子量/游离型分子量，修约至小数点后三位）；统计参数（如干扰率均值、CV%）修约规则：均值保留小数点后一位，CV% = (标准差 / 均值) × 100%，修约至整数一位。逻辑闭环：取数 → 折算（若适用） → 计算 → 对比 → 判定描述。

### **第二部分：通用模板**
* **定义**：可直接由 AI Generator 执行的脚本指令。  
1. 从附件/原始记录（如表5）提取内标对待测物的干扰数据，包括峰面积变化值和基准值。  
2. 若VP指定折算（如游离型/盐型），应用公式：游离型干扰率 = 盐型干扰率 × {{Salt_to_Free_Factor}}（{{Salt_to_Free_Factor}} = 盐型分子量 / 游离型分子量，从VP提取，修约至三位小数）。  
3. 计算干扰率：{{Calculated_Interference_Rate}} = (峰面积变化 / 基准峰面积) × 100%（修约至小数点后一位）；计算均值{{Mean_Interference}}（修约至小数点后一位）和CV%{{CV_Percent}} = (标准差 / 均值) × 100%（修约至整数一位）。  
4. 对比VP接受标准{{Acceptable_Interference_Limit}}%：若{{Calculated_Interference_Rate}}在±{{Acceptable_Interference_Limit}}%内，判定“符合”；否则，列出偏差值并说明。  
5. 填充报告语句：按验证方案测定，内标对待测物的干扰结果见表{{Table_Number}}。结果表明，内标对待测物的干扰{{Judgment_Phrase}}接受标准。（{{Judgment_Phrase}} = “符合”或“部分不符合，具体偏差见表{{Table_Number}}”）。若无数据：按验证方案测定，无干扰测试执行，结果符合接受标准。
