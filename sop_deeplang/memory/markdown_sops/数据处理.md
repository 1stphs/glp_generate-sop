# LC-MS/MS数据处理

## 目的
规定LC-MS/MS生物分析方法验证试验数据处理的标准化流程，确保数据完整性和可追溯性。

## 适用范围
适用于食蟹猴EDTA-K2血浆样品中待测物A（游离型）反义链和正义链浓度测定的LC-MS/MS方法验证数据处理。

## 操作步骤
1.1 从实验室信息管理系统导出原始数据文件，包括：
   - 仪器原始数据文件
   - 样品序列信息
   - 系统适用性数据
   
1.2 确认数据完整性：
   - 检查所有标准曲线点数据存在性
   - 确认质控样品数据完整性
   - 验证系统适用性数据符合要求

1.3 建立数据处理模板：
   - 导入表7标准曲线参数汇总数据
   - 导入表8标准曲线回算浓度数据
   - 导入表9批内、批间准确度及精密度数据

1.4 执行标准曲线拟合：
   - 使用验证方案指定的拟合模型
   - 记录回归方程参数（斜率、截距、相关系数）
   - 计算各浓度点回算浓度

1.5 计算质控样品结果：
   - 计算LLOQ、LQC、MQC、HQC各水平浓度
   - 计算批内、批间准确度和精密度
   - 确认所有QC结果符合接受标准

1.6 汇总分析结果：
   - 整理表1批量样品检测情况
   - 汇总表10质控样品浓度数据
   - 生成方法学验证参数统计表

1.7 数据审核：
   - 双人审核所有计算结果
   - 确认数据与原始文件一致性
   - 标记任何不符合接受标准的结果

## 文档记录
记录以下信息至数据处理记录表：

| 记录项目 | 记录内容 | 记录人 | 审核人 | 日期 |
|---------|----------|--------|--------|------|
| 数据文件编号 |  |
| 标准曲线参数 | 斜率、截距、r² |
| QC准确度 (%) | LLOQ/MQC/HQC |
| QC精密度 (%CV) | LLOQ/MQC/HQC |
| 批内/批间统计 | 准确度/精密度 |
| 审核确认 | 符合/不符合标准 |

## 异常处理
1. **标准曲线不符合接受标准**：重新检查仪器数据，必要时重新分析
2. **QC结果超差**：追溯批次记录，判定是否为系统性偏差，报告质量部门
3. **数据丢失或损坏**：立即通知IT部门恢复备份，暂停后续处理
4. **计算结果异常**：验证计算公式，使用已验证Excel模板重新计算
5. **审核不一致**：组织三方会审，记录会议纪要并归档
