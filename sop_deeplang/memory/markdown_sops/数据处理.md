## 一、核心填写规则
- 严格区分静态描述（如系统列表、公式声明）和动态数据（如修约结果），静态部分直接从方案复制，动态部分基于实际计算提取并修约。
- 数据修约必须遵循以下保留位数规则，确保所有统计参数一致性：准确度（%Accuracy）保留小数点后1位；精密度（%CV）保留小数点后1位；偏差（%Bias）保留小数点后1位；浓度值保留小数点后3位；峰面积比保留小数点后6位。修约前计算完整性由Excel 100%审核验证。
- 对于游离型与盐型数据折算，必须定义转换因子（如分子量比{{Molecular_Weight_Free / Molecular_Weight_Salt}}），计算步骤为：(1) 提取游离型原始浓度；(2) 乘以转换因子得到盐型等效浓度；(3) 修约至指定位数；(4) 若超出接受标准，添加分支描述“不符合，需调查”。
- 强调数值差异说明：若表格汇总与单独计算结果因有效数位取舍而异，必须声明“差异不影响数据完整性和连续性”。
- 公式声明为静态文本，直接列出准确度、精密度、偏差的通用计算公式（无具体数值），并在动态部分应用至实际数据。
- 处理无数据情况：若无偏差或计算结果，仅声明“无偏离，所有数据符合方案要求”。

## 二、通用标准化模板
撰写“数据处理”章节时，严格按照以下结构和指令执行：

1. **有效数位差异声明**：  
   由于不同的程序有效数位的取舍方法有一定的差异，某些表格中的数值和单独计算结果相比可能会有所差别，但数据的完整性和连续性并不会受其影响。

2. **计算机化系统列表**：  
   从方案附件或记录中提取系统信息，填充以下表格（若无更新，直接复制静态列表）：  
   | 软件名称 | 版本号 | 用途 |  
   | --- | --- | --- |  
   | Sciex Analyst | {{Version_1}} & {{Version_2}} | 数据采集及处理 |  
   | Watson LIMS系统 | {{LIMS_Version}} | 回归计算和报告 |  
   | Microsoft Excel | {{Excel_Version}} | 统计计算，计算过程经过100%审核 |  
   指令：从原始记录提取版本号{{Version_1}}、{{Version_2}}、{{LIMS_Version}}、{{Excel_Version}}，确保用途描述匹配方案。

3. **计算公式声明**：  
   方法准确度、精密度及偏差由以下公式计算：  
   - 准确度：{{Formula_Accuracy}}（例如，(测得浓度 / 理论浓度) × 100%）  
   - 精密度：{{Formula_Precision}}（例如，(标准差 / 均值) × 100%）  
   - 偏差：{{Formula_Bias}}（例如，((测得浓度 - 理论浓度) / 理论浓度) × 100%）  
   指令：静态复制公式框架，从方案提取通用表达式；若有游离/盐型折算，插入计算链：提取{{Raw_Free_Concentration}} → 计算{{Salt_Equivalent = Raw_Free_Concentration × (Molecular_Weight_Free / Molecular_Weight_Salt)}} → 修约至小数点后3位。

4. **数据修约方式表格**：  
   | 项目 | 保留位数 |  
   | --- | --- |  
   | 准确度（%Accuracy） | 小数点后1位 |  
   | 精密度（%CV） | 小数点后1位 |  
   | 偏差（%Bias） | 小数点后1位 |  
   | 浓度值 | 小数点后3位 |  
   | 峰面积比 | 小数点后6位 |  
   指令：静态复制表格；动态应用时，从Excel计算结果提取{{Calculated_Accuracy}}、{{Calculated_CV}}、{{Calculated_Bias}}，修约后对比方案标准{{Acceptance_Criteria}}（如±15%），若符合，描述“所有参数在接受范围内”；若不符合，添加“{{Parameter}}值为{{Repaired_Value}}，超出标准{{Acceptance_Criteria}}，建议进一步验证”。

5. **分支逻辑处理**：  
   - 若所有数据符合：结束声明“数据处理符合GLP要求，无偏离”。  
   - 若有折算或偏差：插入“经{{Molecular_Weight_Free / Molecular_Weight_Salt}}因子折算后，{{Repaired_Salt_Concentration}}符合标准”。  
   - 无数据时：输出“本节无额外数据处理需求，所有计算基于方案标准执行”。
