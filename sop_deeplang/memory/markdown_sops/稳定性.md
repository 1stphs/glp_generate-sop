### **第一部分：核心准则**
* **定义**：该章节撰写时必须死守的"法律"。

1. **数据提取与计算规则**：从指定Excel表中提取峰面积比或浓度数据，计算平均值（Mean）、标准差（SD）、变异系数（CV% = (SD/Mean) × 100，保留1位小数）、准确度（% = (测定值/理论值) × 100，保留1位小数）和偏差（% = ((稳定性样品Mean - 参比Mean)/参比Mean) × 100，保留1位小数）。对于所有统计参数，修约规则为：平均值保留3位小数，SD和CV%保留3位小数后四舍五入至1位小数，差异/准确度保留1位小数。放置时间计算为结束时间减开始时间，单位为小时或天，保留1位小数。
2. **接受标准抽象**：稳定性结论基于偏差≤±15.0%（或准确度85.0%-115.0%）和CV%≤15.0%。如果数据符合，描述为“稳定至少{{Placement_Time}}”；若不符合，添加“未达标，偏差{{Deviation_%}}%”并建议重新验证。内标稳定性若无考察，直接声明“符合干扰标准”。
3. **分支逻辑**：无数据时，使用声明式表述“按验证方案测定，结果符合接受标准”；有数据时，按类型（储备液/工作液/质控样品）分段报告，引用对应表号，确保结论覆盖所有子类型（短期/长期/冻融等）。
4. **通用化要求**：结论不依赖具体数值，使用占位符如{{Stability_Type}}、{{Placement_Time}}、{{Deviation_%}}；确保描述逻辑闭环：提取→计算→对比→判定。

* **定义**：可直接由 AI Generator 执行的脚本指令。

**Prompt指令脚本**：

2. 对于质控样品稳定性（e.g., 表18_ST），提取LQC/HQC浓度{{Nominal_Concentration}}（ng/mL）、测定值（6个平行，计算Mean、SD、CV%、准确度% = (Mean/Nominal) × 100）；修约：Mean保留3位小数，SD/CV%保留至1位小数，准确度保留1位小数。
3. 对比接受标准：若|{{Deviation_%}}| ≤ 15.0% 且 CV% ≤ 15.0%，判定为稳定；否则，标记“不符合”。
4. 填充报告模板：
