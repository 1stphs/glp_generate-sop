## 一、核心填写规则
1. **二元差异定位**：静态文本直接从验证方案（VP）复制，包括方法描述（如仪器、溶剂、配制步骤）和通用安全要求（如防护用品、剩余标准品归还）；动态数据从原始记录（如实验日志、Excel数据表）提取，包括实际测定结果、计算值（如偏差%、CV%、R²）和判定结论。
2. **接受标准抽象**：报告中所有判定须基于VP中定义的红线（如偏差±15.0%、CV≤15.0%、R²≥0.9800），使用占位符表示（如{{Deviation_Range}} = ±15.0%，{{CV_Limit}} = 15.0%）。若实际数据不符合，触发分支逻辑：记录不合规原因、影响评估（如残留因子计算），并建议优化或重测。
3. **计算逻辑原子化**： 
   - 偏差计算：(测定浓度 - 标示浓度) / 标示浓度 × 100%，修约至小数点后一位（e.g., {{Calculated_Deviation}} = ±X.X%）。
   - CV%计算：(标准差 / 均值) × 100%，修约至小数点后一位；残留因子 = 1 / (双空白峰面积 / 前ULOQ峰面积)，修约至小数点后三位；影响率 = (上一样品峰面积 / 残留因子 / 下一峰面积) × 100%，修约至小数点后一位。若有折算因子（如反义链/正义链浓度比例0.463:0.537），在提取时明确公式：正义链浓度 = 反义链浓度 × (0.537 / 0.463)，修约至小数点后三位。
   - 标准曲线回归：权重1/X²，R²修约至小数点后四位；回算偏差针对LLOQ为±20.0%，其他±15.0%，修约至小数点后一位。
   - 统计参数精度：所有浓度单位为ng/mL，修约至小数点后三位；峰面积比值无单位，修约至小数点后四位。确保批内/批间评估覆盖至少3批、6重复/浓度。
4. **分支处理**：若无偏离或数据为空，报告简述“符合VP要求，无偏差”；若不符合，列出影响（如“残留影响率>5%，样品重测”），并引用SOP-EXO-ABA-004评估。
5. **结构对齐**：报告开头固定包含试验名称、编号、负责人、机构、委托方、签发日期、总页数；后续章节按VP顺序（如系统适用性、残留、选择性）填充动态内容，确保逻辑闭环（取数→计算→对比→描述）。

2. **计算与修约**：
   - 计算偏差、CV%、R²、残留因子/影响率，按核心规则原子化步骤执行，所有输出修约一致（浓度至小数点后三位，百分比至小数点后一位，四位数如R²至小数点后四位）。

**系统适用性**  
系统适用性考察是在分析批开始前进行以确保系统正常运转。处理至少1份系统适用性样品并连续进样至少3次，以最后有效的3次进样来评估系统适用性。  
结果：反义链保留时间均值2.15 min，CV 1.2%（≤5.0%）；峰面积比值CV 3.5%（≤15.0%）。正义链保留时间均值2.18 min，CV 1.4%（≤5.0%）；峰面积比值CV 4.1%（≤15.0%）。  
结论：系统适用性符合接受标准。  

**残留**  
残留通过在定量上限（ULOQ）样品后面进样双空白样品（不含待测物和内标）来进行评估。  
结果：双空白样品中反义链响应为LLOQ均值的8.5%（≤20.0%），内标响应为LLOQ内标均值的2.3%（≤5.0%）。残留因子0.923，影响率3.2%（≤5.0%）。  
结论：残留符合接受标准，无需额外处理。
