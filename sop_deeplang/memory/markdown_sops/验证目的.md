### **第一部分：核心准则**
* 该章节作为报告引言，必须概述验证目的和覆盖的验证参数，确保逻辑覆盖所有必要验证要素（如系统适用性、残留、选择性等），以支持非临床试验生物分析。
* 内容必须基于协议中指定的分析方法和样品类型，保持客观描述，不引入具体数据或偏差，仅列出验证参数列表。
* 若无额外偏差或特殊情况，使用声明式表述确认验证范围完整。

### **第二部分：通用模板**
撰写报告引言时，按照以下指令执行：
1. 从协议中提取分析方法名称（{{Method_Code}}）、动物模型和样品类型（{{Sample_Type}}）、目标化合物（{{Compound_Name}}，{{Form_Type}}）。
2. 列出所有验证参数，使用固定列表格式：系统适用性、残留、选择性、特异性、基质效应、标准曲线及范围、批内/批间准确度及精密度、提取回收率、稀释可靠性、稳定性、溶血效应和分析批最大进样数（若协议有更新，替换为{{Validation_Parameters_List}}）。
3. 构建句子：本研究的目的是验证{{Method_Description}}，包括{{Validation_Parameters_List}}等，以支持非临床试验生物分析。
4. 确保表述简洁，不超过一段。
