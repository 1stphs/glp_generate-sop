### **第一部分：核心准则**
1. **合规性底线**：严格遵循FDA 21 CFR Part 11/GLP要求，确保所有数据引用可追溯至原始记录（Worksheets/附件），计算过程透明、可验证，保留位数按GLP标准（通常小数点后2位）。
2. **二元差异定位**：静态文本直接从VP（验证方案）复制；动态数据从VR原始记录提取，包括检测次数、平均值（Mean）、CV%、不确定度（Uncertainty）、日期范围。
3. **判定逻辑**：对比VP接受标准（如Mean偏差≤±15.0%、CV≤X%）与实际结果，生成客观描述。若符合，声明“满足接受标准”；若不符合，报告实际值并说明偏差原因（引用Worksheets）。
4. **计算原子化**：Mean = Σ结果/次数；CV% = (SD/Mean)×100；不确定度按Worksheets公式计算（k=2，95%置信区间），修约至小数点后2位。
5. **分支处理**：若数据不符合，添加“建议复测”条款；确保描述中立、不推断。
