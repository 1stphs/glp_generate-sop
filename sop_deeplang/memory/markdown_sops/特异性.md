## 一、核心填写规则
1. **特异性验证定义**：特异性章节必须评估分析方法对目标分析物（如反义链）的选择性，区分干扰峰（如空白基质、游离型/盐型杂质）与目标峰的响应。接受标准基于协议中定义的峰纯度阈值（如目标峰无干扰峰重叠，峰面积差异≤±2.0%）。
2. **数据来源与计算逻辑**：从原始色谱数据（Excel附件）提取目标峰面积（{{Target_Peak_Area}}）、干扰峰面积（{{Interference_Peak_Area}}），计算特异性指标：相对响应差异 = (|{{Target_Peak_Area}} - {{Interference_Peak_Area}}| / {{Target_Peak_Area}}) × 100%。结果四舍五入保留1位小数。若涉及游离型/盐型折算，使用公式：盐型含量 = {{Free_Form_Content}} × ({{Molecular_Weight_Salt}} / {{Molecular_Weight_Free}})，其中分子量比从协议中获取，保留2位小数。所有计算须确保无偏离（实际值符合协议阈值），否则记录偏差并说明影响。
3. **分支逻辑**：若数据符合标准，声明“特异性符合要求”；若不符合，描述偏差原因、影响及补救措施。无数据时，声明“本方法特异性验证未进行，或无偏离”。
4. **通用化要求**：避免具体数值，使用占位符；确保逻辑闭环，包括数据提取→计算→对比→结论。
