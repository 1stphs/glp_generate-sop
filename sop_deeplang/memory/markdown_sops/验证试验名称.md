### **第一部分：核心准则**
1. **二元数据区分**：静态文本（如“游离酸含量”、“≤2.0%”）直接从验证方案（VP）复制；动态数据（如实际测定值、日期、判定结果）从原始记录（如附件HPL C色谱图、标准曲线数据）提取。
2. **合规性判定逻辑**：比较实际最高峰面积占比与VP接受标准（≤2.0%）。若≤2.0%，判定"符合"；若>2.0%，判定"不符合"并说明偏差原因。保留小数点后1位。
3. **计算原子化**：游离酸含量（%）=（游离酸峰面积 / 总峰面积）×100。总峰面积为所有主要杂质峰+主组分峰面积总和。
4. **完整性要求**：必须包含日期范围、结论性陈述及数据来源引用。处理边界情况：如无游离酸峰，报告"<0.1%"并判定符合。
5. **GLP合规**：所有数据追溯至原始记录，确保可审核性。
