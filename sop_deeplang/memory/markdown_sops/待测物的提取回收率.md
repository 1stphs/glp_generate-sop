* 每个浓度水平的回收率平均值必须在{{Acceptable_Recovery_Range}}（例如85%-110%）范围内，变异系数（%CV）不超过{{Acceptable_CV_Limit}}（例如≤15%）；总体%CV不超过{{Overall_CV_Limit}}（例如≤15%）。
* 从表11提取每个浓度水平的平均回收率（%）和%CV值；若实际值符合接受标准，描述为“结果表明...平均值在{{Min_Recovery}}%~{{Max_Recovery}}%之间，变异系数（%CV）均不大于{{Max_Individual_CV}}%；总体变异系数（%CV）为{{Overall_CV}}%”；若不符合，需标注偏差并说明原因。
* 计算步骤：1. 从原始数据计算每个浓度的平均回收率 = (∑峰面积比)/n；2. 计算%CV = (标准差/平均值)×100，保留1位小数；3. 汇总总体%CV。
