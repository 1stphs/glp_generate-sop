## 一、核心填写规则
1. **数据来源与判定依据**：从验证方案（VP）中提取待测物对内标的干扰接受标准（如干扰率阈值，例如 ≤ ±15.0%）；若VP无具体标准，默认报告符合性声明基于实际测试结果。若存在游离型/盐型待测物，识别并应用折算因子公式：游离型含量 = 盐型含量 × (分子量_游离型 / 分子量_盐型)，保留小数点后两位；干扰率计算公式：干扰率 (%) = [(有待测物峰面积 - 无待测物峰面积) / 无待测物峰面积] × 100，修约至小数点后一位。
2. **合规性检查**：比较实际干扰结果（如{{Interference_Rate}}）与VP接受标准；若符合，生成正面判定语；若不符合，报告实际值并注明偏离原因。无数据时，使用标准声明句，避免具体数值。
3. **分支逻辑**：若测试无偏离，直接引用表号并声明符合；若有干扰数据，列出关键指标（峰面积比、干扰率）并确保描述闭环（来源-计算-结论）。
4. **通用化原则**：禁止硬编码具体值，使用占位符如{{Table_Number}}、{{Interference_Rate}}；确保描述商务化、中性，避免主观词语。

## 二、通用标准化模板
**撰写指令**：  
1. 从附件/原始记录提取干扰测试数据，包括有/无待测物峰面积；若适用，应用折算因子公式计算调整后值：{{Adjusted_Content}} = {{Raw_Salt_Content}} × ({{MW_Free}} / {{MW_Salt}})（保留两位小数）。  
2. 计算干扰率：{{Interference_Rate}} = [( {{Peak_With_Analyte}} - {{Peak_Without_Analyte}} ) / {{Peak_Without_Analyte}} ] × 100（修约至一位小数）。  
3. 比较{{Interference_Rate}}与VP标准{{Acceptance_Criterion}}（例如 ≤ ±{{Threshold}}%）；生成描述：  
   “按验证方案测定，待测物对内标的干扰结果见表{{Table_Number}}。结果表明，待测物对内标的干扰{{Judgment_Phrase}}。”  
   - 若符合：{{Judgment_Phrase}} = “符合接受标准”；  
   - 若不符合：{{Judgment_Phrase}} = “不符合接受标准，实际干扰率为{{Interference_Rate}}%，偏离原因{{Deviation_Reason}}”。  
4. 若无数据：使用默认句“按验证方案测定，待测物对内标的干扰结果见表{{Table_Number}}。结果表明，待测物对内标的干扰符合接受标准。”  
5. 排版：置于章节正文，表号引用VP对应表格；确保句子简洁，一段不超过50字。
