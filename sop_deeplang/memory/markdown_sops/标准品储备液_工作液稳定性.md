### **第一部分：核心准则**
评估标准品储备液和工作液的稳定性时，必须基于验证方案中规定的测试条件（室温白光和超低温冰箱 -70~-90 ºC），从原始数据表中提取实际测量结果（如浓度变化百分比、存储时长）。判定逻辑：如果实际稳定性时长达到或超过预设阈值，则描述为“稳定至少{{Stability_Hours/Days}}”；若不符合，需报告偏差并说明影响。计算步骤：1. 从数据表计算平均浓度变化（Mean % Change = (最终浓度 - 初始浓度)/初始浓度 × 100，保留1位小数）；2. 比较变化与接受标准（通常≤±{{Acceptance_Limit}}%）；3. 确定最大稳定时长（从数据点中取最大符合时长值，单位为小时/天）。所有描述需引用对应数据表，并确保表述客观、中性。

### **第二部分：通用模板**
从验证方案中确认测试条件后，执行以下步骤撰写：  
1. 提取储备液数据：从表{{Table_Reserve_1}}和表{{Table_Reserve_2}}获取{{Storage_Condition_1}}（室温白光）下的{{Stability_Hours}}和{{Storage_Condition_2}}（超低温冰箱 -70~-90 ºC）下的{{Stability_Days}}。  
2. 计算并判定：若平均浓度变化≤{{Acceptance_Limit}}%，则确认稳定；否则，标记偏差。  
3. 填入模板：  
“标准品储备液稳定性按验证方案测定，结果见表{{Table_Reserve_1}}和表{{Table_Reserve_2}}。结果表明{{Compound_Name}}（游离型）反义链和正义链标准品储备液可在{{Storage_Condition_1}}条件下稳定至少{{Stability_Hours}}，在{{Storage_Condition_2}}稳定至少{{Stability_Days}}。”  
4. 重复步骤1-3于工作液，使用表{{Table_Work_1}}和表{{Table_Work_2}}，调整描述为“工作液”并保持一致格式。  
若无偏差，添加总结句：“所有结果符合接受标准。”
