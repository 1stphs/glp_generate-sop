### **第一部分：核心准则**
在该章节中，如果验证方案（Protocol）中仅提供执行人员信息且无具体试验数据或接受标准，则报告（Report）必须声明“无数据”，以确认无偏差或额外记录。严禁添加任何未在方案中出现的试验细节或计算。动态部分仅限于插入执行人员姓名、联系方式作为归属标识。

### **第二部分：通用模板**
从验证方案（Protocol）提取执行人员信息，包括姓名 `{{姓名}}`、电话 `{{电话}}` 和电子邮件 `{{电子邮件}}`，并在报告中按以下格式呈现个人信息表格。若无其他数据，立即添加声明“无数据”。完整指令：  
| 姓名： | {{姓名}} |  
| --- | --- |  
| 电话： | {{电话}} |  
| 电子邮件： | {{电子邮件}} |
