### **第一部分：核心准则**
1. **数据来源优先级**：优先从原始记录（如附件中的色谱图、峰面积表）提取实际数据，包括批号、检测日期、平均值（Mean）、相对标准偏差（RSD%）、游离酸含量等。协议（62字符）中的接受标准（如RSD ≤ 2.0%、偏差 ±10.0%以内）作为合规判断红线。
2. **合规判定逻辑**： 
   - 计算实际RSD% = (标准差 / 平均值) × 100，保留1位小数。
   - 偏差% = |(测定值 - 标示值)/标示值| × 100，保留1位小数。
   - 若所有RSD ≤ 协议阈值且偏差在限内，判定“符合”；否则列出具体超标项目并说明影响。
3. **通用化描述**：禁止硬编码具体数值，使用 {{Mean_Value}}、{{RSD_Percent}}、{{Storage_Days}} 等占位符。确保描述包含“方法、数据、结论”三元组，符合FDA 21 CFR Part 11完整性要求。
4. **分支处理**：若不符合，客观陈述“实际值 {{Actual_RSD}}% 超出 {{Acceptance_Criteria}}”，不回避事实。
5. **修约规则**：所有百分比保留1位小数，绝对值保留2位小数；日期格式为 YYYY-MM-DD。

### **第三部分：报告示例**
【游离酸含量测定】

根据验证方案（VP）要求，对样品（批号：SOP20240101）进行25℃稳定性考察30天。

1. 数据提取：从附件3（色谱峰面积表）获取以下数据：
   - 各时间点Mean = 峰面积平均值（n=6），RSD = (SD/Mean)×100%。
   - 游离酸含量 = (游离酸峰面积/总酸峰面积 × 100)%。

2. 合规对比：
   | 时间点 | Mean (%) | RSD (%) | 偏差 (%) | 接受标准 |
   |--------|----------|---------|----------|----------|
   | 0天    | 0.12    | 0.8    | 0.0     | RSD ≤ 2.0%，偏差 ≤ ±10.0% |
   | 15天   | 0.13    | 1.2    | 8.3     | RSD ≤ 2.0%，偏差 ≤ ±10.0% |
   | 30天   | 0.14    | 1.5    | 16.7    | RSD ≤ 2.0%，偏差 ≤ ±10.0% |

3. 结论：所有时间点RSD均为0.8%~1.5%（≤ 2.0%），但30天偏差16.7%超出±10.0%限值，**部分不符合接受标准**。建议优化配方以降低游离酸生成。
