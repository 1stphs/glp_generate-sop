# 生物样品预处理标准操作程序 (SOP)

## 目的
规定生物样品预处理过程的标准操作流程，确保21 CFR Part 11和GLP法规（21 CFR Part 58）合规性，适用于多种LC-MS/MS分析方法验证。

## 适用范围
适用于LC-MS/MS生物分析方法验证中的生物基质样品（血浆、血清、尿液、全血等）预处理，包括蛋白沉淀、离心、氮吹及复溶操作。

## 操作步骤

**1. 样品准备与解冻**  
1.1 从指定温度冰箱（-20°C或-80°C，仪器编号：LIMS系统查询）取出样品批次  
1.2 记录解冻开始时间，在冰浴（2-6°C）中解冻至完全液化  
1.3 涡旋混匀15±5秒，确保样品均匀无沉淀  
1.4 视觉检查样品完整性，记录溶血程度（无/轻度/明显）  

**2. 蛋白沉淀（可配置体积比）**  
2.1 准确移取[100-200] μL样品至合适容器的离心管（规格：1.5/2.0 mL）  
2.2 按验证方案比例加入冰冷沉淀剂（乙腈/甲醇，3:1至4:1 v/v）  
   - 记录实际加入体积和试剂批号  
2.3 涡旋混匀30±5秒  
2.4 超声浴处理（频率37-40 kHz，5±1分钟，温度≤25°C）  

**3. 离心分离**  
3.1 使用经校准冷冻离心机（仪器编号：LIMS系统查询）  
3.2 设置参数（按验证方案）：  
   ```
   温度：2-6°C  
   转速：12,000-15,000 rpm  
   时间：10-15分钟  
   ```
3.3 平衡转子，径向对称放置样品管（R.C.F. ≥15,000g）  
3.4 启动程序，离心完成后5分钟内转移上清液  

**4. 上清液处理与复溶**  
4.1 小心转移[150-250] μL上清液至氮吹专用管  
4.2 氮气吹干（温度35-45°C，氮气流速15-25 L/min，约20-40分钟）  
4.3 用[50-100] μL复溶液（按验证方案：甲醇-水50:50至90:10 v/v）复溶  
4.4 涡旋30±5秒，超声1分钟，确保完全溶解  
4.5 转移至 autosampler 小管，4°C保存待分析

## 异常处理

**1. 温度偏差**  
- 离心温度>8°C：暂停批次，记录实际温度，评估重做需求  
- 氮吹温度>50°C：终止操作，重新准备受影响样品  

**2. 样品质量问题**  
```
溶血程度：轻度-记录批注；明显-备用样品替代
沉淀不完全：延长超声2分钟，重新评估
上清液浑浊：记录体积，浓度校正系数x1.2
```

**3. 仪器故障流程**  
```
1. 立即停止操作，贴"设备故障"标签
2. 改用LIMS系统查询的备用合格仪器
3. 填写"仪器故障报告"，24小时内通知QA
4. 重新验证备用仪器运行参数
```

**4. 偏差管理**  
```
所有偏差必须：
□ 填写"实验室偏差报告"（LAB-DEV-001）
□ 24小时内QA审核
□ 影响数据完整性则批次重做
□ CAPA计划48小时内制定
```

**5. 紧急情况**  
```
电力故障：立即转移至备用电源冰箱
化学品泄漏：按"化学品泄漏应急SOP"执行
人员暴露：立即就医，报告EHS部门
```
