# 生物分析方法验证试验归档SOP

## 目的
规定LC-MS/MS法生物分析方法学验证试验原始记录和相关文件的标准归档流程，确保数据完整性和可追溯性。

## 适用范围
适用于SS25071BV02验证试验编号下LC-MS/MS法测定食蟹猴EDTA-K2血浆样品中待测物A（游离型）反义链和正义链浓度的方法学验证归档。

## 操作步骤
1.1 确认归档文件完整性清单：
    - 验证方案（SS25071BV02）
    - 分析方法（附录II）
    - 原始色谱图谱数据（附图目录）
    - 所有试验数据表（表1-表23）
    - 质检报告（附录III）
    - 仪器使用日志

1.2 整理纸质文件：
    - 按附表目录顺序编号装订
    - 每份文件右上角标注页码
    - 封面标注：验证编号SS25071BV02、归档日期、归档人

1.3 电子数据归档：
    - 原始数据文件命名格式：SS25071BV02_[日期]_[文件类型]
    - 将所有电子文件复制至指定服务器路径
    - 建立电子文件目录结构与纸质文件一致

1.4 填写归档登记表：
    - 记录归档总页数
    - 记录电子文件总容量
    - 双人核对文件完整性

1.5 归档文件存放：
    - 纸质文件存入指定GLP档案柜
    - 电子文件备份至两套独立存储系统
    - 记录档案柜编号和电子存储路径

## 异常处理
1. 文件缺失：立即追溯文件流转记录，联系相关人员补齐，记录补齐原因和日期
2. 电子数据损坏：使用备份文件恢复，验证数据一致性后重新归档
3. 归档文件不完整：暂停归档，通知QA部门，完成补齐后重新执行归档流程
4. 任何异常情况须在24小时内书面报告质量保证部门
