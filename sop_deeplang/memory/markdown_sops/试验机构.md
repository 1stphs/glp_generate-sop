### **第一部分：核心准则**
在报告的“测试设施信息”章节中，必须严格按照 GLP 要求，提供测试设施的标准化基本信息，包括公司名称、中英文对照、地址（若有多个地址逐行列出）、邮编和联系电话。信息来源于协议或官方记录，确保准确无误、无动态数据计算，仅作为静态声明使用。若无额外细节，仅报告核心字段，不添加解释性文字。

### **第二部分：通用模板**
使用以下表格格式报告测试设施信息，直接复制静态字段，不涉及数据提取或计算：

| 名称： | {{Company_Name_CN}} |
| --- | --- |
|  | （{{Company_Name_EN}}） |
| 地址： | {{Address_Line_1}} |
|  | {{Address_Line_2}} |
| 邮编： | {{Postal_Code}} |
| 电话： | {{Phone_Number}} |

指令：将 {{Company_Name_CN}}、{{Company_Name_EN}} 等占位符替换为协议中指定的固定值。若地址超过一行，按顺序填充。
