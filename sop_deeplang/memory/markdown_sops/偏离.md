### **第一部分：核心准则**
在撰写该章节时，必须确认整个实验过程严格遵守验证方案（Protocol）、变更记录、分析方法以及机构标准操作规程（SOP）。核心判定标准为：无任何偏离发生。若存在偏离，必须记录并评估其对实验结果的影响，但本章节仅适用于无偏离场景。动态检查点：从实验记录中提取偏离日志，若为空，则输出声明语句；严禁编造偏离信息。

### **第二部分：通用模板**
执行以下指令生成报告片段：
1. 从实验原始记录（包括方案、变更日志、方法文档和SOP合规检查表）中提取偏离相关信息。
2. 若无任何偏离记录，填入以下模板语句：`本实验过程中无方案及变更、方法和本机构SOP的偏离发生。`
3. 若检测到偏离，切换至分支逻辑：记录偏离描述、发生时间、影响评估，并以`本实验过程中发生以下偏离：{{Deviation_Description}}。评估结果：{{Impact_Assessment}}。`格式描述。
4. 确保语句简洁、正式，置于章节独立段落中，无需额外解释。

### **第三部分：报告示例**
本实验过程中无方案及变更、方法和本机构SOP的偏离发生。
