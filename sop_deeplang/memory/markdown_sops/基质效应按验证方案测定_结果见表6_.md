### **第一部分：核心准则**
* **定义**：该章节撰写时必须死守的"法律"。

基质效应试验评估生物基质对分析方法准确度的影响，使用低质量对照（LQC，标称浓度{{Nominal_LQC}} ng/mL）和高质量对照（HQC，标称浓度{{Nominal_HQC}} ng/mL）在至少3个生物基质批号中进行平行测定。每个批号计算3个平行样本的平均值（Mean）、标准差（SD）、变异系数（%CV = (SD / Mean) × 100）、准确度%（Accuracy% = (Mean / Nominal) × 100）和准确度偏差%（Bias% = Accuracy% - 100）。所有数值修约规则：浓度值保留3位小数，百分比值（%CV、Accuracy%、Bias%）保留1位小数；若计算结果超出接受标准（{{Acceptance_Criterion}}，例如Bias%在±{{Max_Bias}}%以内，%CV < {{Max_CV}}%），则判定为不符合，并描述偏差原因及影响；无数据或无偏差时，直接声明“结果表明基质效应符合接受标准”。数据从附件表{{Table_Number}}_ME提取，确保逻辑闭环：提取原始数据 → 计算统计参数 → 对比接受标准 → 生成判定描述。

### **第二部分：通用模板**
* **定义**：可直接由 AI Generator 执行的脚本指令。

1. 从附件{{Table_Number}}_ME提取数据：针对每个生物基质批号{{Matrix_Batch}}，获取LQC和HQC的3个平行浓度值{{Conc_1}}、{{Conc_2}}、{{Conc_3}}。
2. 计算统计参数：
   - Mean = ({{Conc_1}} + {{Conc_2}} + {{Conc_3}}) / 3，修约至3位小数。
   - SD = √[Σ({{Conc_i}} - Mean)^2 / (n-1)]，n=3，修约至3位小数。
   - %CV = (SD / Mean) × 100，修约至1位小数。
   - Accuracy% = (Mean / {{Nominal_Conc}}) × 100，修约至1位小数。
   - Bias% = Accuracy% - 100，修约至1位小数。
3. 汇总所有批号的总体Mean、SD、%CV、Accuracy%、Bias%，并对比{{Acceptance_Criterion}}。
4. 生成描述：若所有Bias% ∈ [-{{Max_Bias}}, +{{Max_Bias}}]且%CV < {{Max_CV}}，输出“结果表明基质效应符合接受标准，结果见表{{Table_Number}}。”；否则，输出“基质效应在{{Specific_Batch}}批号的{{LQC_or_HQC}}中{{Deviation_Description}}（实际{{Calculated_Bias}}%），不符合{{Acceptance_Criterion}}，详见表{{Table_Number}}。”。插入表{{Table_Number}}，包含批号、浓度、统计参数列。
