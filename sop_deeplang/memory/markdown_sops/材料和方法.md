# LC-MS/MS法测定食蟹猴EDTA-K2血浆样品中待测物A（游离型）反义链和正义链含量SOP

## 目的
规定LC-MS/MS法测定食蟹猴EDTA-K2血浆样品中待测物A（游离型）反义链和正义链含量标准操作流程，确保分析结果准确可靠。

## 适用范围
适用于LC-MS/MS分析食蟹猴EDTA-K2血浆样品中待测物A（游离型）反义链和正义链含量的方法验证及常规样品分析。

## 操作步骤
**2.1 标准品和内标管理**
1.1 从实验室标准品库房领取待测物A标准品和内标A标准品
1.2 检查标准品RoA（Report of Analysis）和内标A的CoA（Certificate of Analysis）
1.3 记录标准品批号、有效期和储存条件
1.4 使用后将剩余标准品及内标按规定返还库房

**2.2 分析系统准备**
2.1 从实验室仪器管理系统查询可用LC-MS/MS系统编号
2.2 按照仪器操作规程启动LC-MS/MS系统
2.3 运行系统适用性试验

**2.3 储备液配制**
3.1 称取待测物A标准品，溶解于指定溶剂制备储备液
3.2 称取内标A标准品，溶解于指定溶剂制备内标储备液
3.3 记录配制时间、溶剂批号、最终浓度

**2.4 工作液配制**
4.1 从储备液稀释配制待测物A工作液系列
4.2 从内标储备液配制内标工作液
4.3 记录各工作液的稀释倍数和配制时间

**2.5 标曲/质控样品配制**
5.1 按浓度梯度在空白血浆中加入相应体积的待测物工作液和内标工作液
5.2 配制标准曲线样品（至少6个浓度点）和质控样品（LLOQ、LQC、MQC、HQC）

**2.6 样品处理步骤**
6.1 取[体积]食蟹猴EDTA-K2血浆样品至适当容器
6.2 加入内标工作液
6.3 进行蛋白沉淀/液液萃取/固相萃取等处理步骤
6.4 [具体处理参数]
6.5 收集上清液/洗脱液至进样瓶

**2.7 LC-MS/MS分析**
7.1 设置LC-MS/MS仪器参数（具体参数按方法验证确定）
7.2 按序进样：系统适用性→双空白→标准曲线→质控样品→待测样品
7.3 每个分析批次不超过[最大进样数]

## 异常处理
1. **系统适用性不符合**：停止进样，检查仪器状态，重新运行系统适用性试验
2. **质控样品超出接受标准**：暂停样品分析，报告质量保证部，重新配制质控样品
3. **标准曲线r²<接受标准**：重新配制标准曲线工作液，重新配制标曲样品
4. **仪器故障**：立即停止分析，记录故障现象，通知仪器管理员维修
5. **稳定性不符合**：按验证方案3.11.5处理办法执行，报告验证负责人
6. **所有异常情况须记录偏差报告单，经质量部门批准后继续或终止试验**
