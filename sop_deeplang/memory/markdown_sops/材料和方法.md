# 实验室样品预处理标准操作程序（SOP）

## 目的
规定实验室样品预处理的标准操作流程，确保符合验证方案、GLP报告要求及21 CFR Part 58和21 CFR Part 11的数据完整性与可追溯性。

## 适用范围
适用于LC-MS/MS、HPLC、GC-MS等分析方法前处理的所有生物基质（血浆、尿液、组织）、化学标准品、QC样品及环境样品。

## 操作步骤

### 1. 仪器准备与确认
1.1 从LIMS/仪器管理系统查询当前可用离心机编号并记录  
1.2 确认仪器校准状态（最近校准日期<12个月，校准证书编号记录）  
1.3 清洁离心机转子和样品放置区域（70%乙醇擦拭）

### 2. 样品预处理参数设置
2.1 根据验证方案和GLP报告设置离心参数：  
    - **温度**：4°C（±2°C）  
    - **转速**：12000rpm（±5%）  
    - **时间**：10分钟（±30秒）  
2.2 验证参数设置显示准确，拍照存档（文件名：仪器编号_日期_批次号.jpg）  
2.3 检查转子平衡，对称放置样品管（对角线位置配重）

### 3. 离心操作执行
3.1 启动离心程序，确认参数锁定后关闭机盖  
3.2 等待离心完成，转子完全停止（约2分钟）  
3.3 按实验室安全规程打开机盖，检查样品管完整性  
3.4 转移样品至指定温度暂存（4°C冰盒或-20°C冰箱）

## 异常处理

### **常见异常及处理流程**：

| 异常类型 | 判定标准 | 立即处理措施 | 文档要求 |
|----------|----------|--------------|----------|
| **仪器报警** | 任何报警代码 | 停止操作，记录报警代码，通知仪器管理员 | 偏差报告+维修记录 |
| **参数偏差** | 温度±2°C外，转速±5%外，时间±30秒外 | 暂停批次，重新校准，重新处理样品 | 偏差报告+重处理记录 |
| **样品异常** | 破裂、泄漏、浑浊 | 隔离区域，按生物安全Level 2清理，重新制备 | 污染事件报告 |
| **电力故障** | 离心中途断电 | 废弃该批次样品，重新制备 | 设备故障报告 |

### **偏差处理通用流程**：
```
1. 立即停止相关操作 → 2. 隔离受影响样品 → 3. 填写"实验室偏差报告" 
→ 4. QA部门24小时内审批 → 5. 制定纠正措施 → 6. 批次放行
```

**注意**：任何偏差必须追溯至验证方案批准参数，偏差率>5%需启动CAPA程序。
