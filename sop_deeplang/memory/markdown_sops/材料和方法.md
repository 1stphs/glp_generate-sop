## 一、核心填写规则
- **内容定位**：该章节用于概述分析方法的关键元素，包括方法编号、样品类型、提取方式、检测技术、内标及仪器模式，确保描述忠实于验证方案（VP）中规定的方法细节，不引入任何未在VP中明确的内容。
- **数据提取原则**：从VP中直接复制静态文本（如方法编号、仪器类型），动态元素（如样品体积、溶剂）需从原始记录或VP附件中提取；若VP中无具体数值，使用通用占位符描述。
- **逻辑闭环**：描述必须完整覆盖“仪器、软件、溶剂/试剂、基质、配制/处理步骤”，并以“详见附录II”等交叉引用结束；若无偏差，保持客观概述语气，无需添加判定语。
- **分支处理**：若方法有偏差（如仪器变更），在描述后添加“无偏离”或“偏差说明：{{Deviation_Description}}”；否则，默认“无偏离”。
