### 第一部分：核心准则
* **定义**：该章节撰写时必须死守的"法律"。
- 回收率计算基于峰面积比：提取回收率（%）= (提取后峰面积比 / 未提取峰面积比) × 100，修约至小数点后一位；平均值修约至小数点后一位；标准差（SD）修约至小数点后一位；变异系数（CV%）= (SD / 平均值) × 100，修约至小数点后一位。
- 总体回收率% 为各浓度平均回收率的平均值，修约至小数点后一位；总体CV% 为各浓度平均回收率的SD/平均值×100，修约至小数点后一位。
- 合规判定：每个浓度的平均回收率应在 {{Acceptable_Range_Min}}% ~ {{Acceptable_Range_Max}}% 范围内（默认范围基于方案，若方案无数据则默认±15%变异可接受），CV% ≤ {{Max_CV}}%（默认15%）；总体CV% ≤ {{Overall_Max_CV}}%（默认15%）。若不符合，报告中添加“未达标”说明并追溯原因。
- 内标回收率：若使用稳定同位素标记内标，直接参照待测物回收率，无需独立计算。
- 数据来源：从原始记录（如表11）提取峰面积比、计算回收率；处理分支：若无数据，声明“按方案执行，无偏差”；若不符合标准，添加判定语如“回收率超出范围，建议优化提取步骤”。
- 逻辑闭环：取数 → 计算（原子步骤：逐个计算回收率 → 求平均/SD/CV → 总体统计） → 对比方案标准 → 描述结果（包括范围、CV、总体变异）。

### 第二部分：通用模板
* **定义**：可直接由 AI Generator 执行的脚本指令。
1. 从原始记录（如 {{Table_ID}}，e.g., 表11）提取数据：对于低浓度（LQC）、中浓度（MQC）、高浓度（HQC），分别获取每个重复的“提取后峰面积比”和“未提取峰面积比”。
2. 计算回收率：对于每个重复，回收率（%）= (提取后峰面积比 / 未提取峰面积比) × 100，修约至小数点后一位；计算每个浓度的平均回收率（{{Mean_Recovery_LQC}}%、{{Mean_Recovery_MQC}}%、{{Mean_Recovery_HQC}}%）、SD（修约至小数点后一位）、CV% = (SD / 平均回收率) × 100（修约至小数点后一位）。
3. 计算总体统计：总体回收率% = ({{Mean_Recovery_LQC}} + {{Mean_Recovery_MQC}} + {{Mean_Recovery_HQC}}) / 3，修约至小数点后一位；总体CV% = [SD of ({{Mean_Recovery_LQC}}, {{Mean_Recovery_MQC}}, {{Mean_Recovery_HQC}}}) / 总体回收率] × 100，修约至小数点后一位。
4. 判定合规：若所有平均回收率在 {{Acceptable_Range_Min}}% ~ {{Acceptable_Range_Max}}% 且 CV% ≤ {{Max_CV}}%，则描述为“符合”；否则添加“不符合，{{具体偏差}}”。
5. 填入报告模板：
   - 待测物的提取回收率按验证方案测定，结果见{{Table_ID}}。结果表明，以峰面积比计算的本方法测定低、中、高浓度待测物每一浓度的提取回收率平均值在{{Min_Mean_Recovery}}%~{{Max_Mean_Recovery}}%之间，变异系数（%CV）均不大于{{Max_CV_Observed}}%；总体变异系数（%CV）为{{Overall_CV}}%。
   - 若内标适用：内标的提取回收率由于使用稳定同位素标记内标，参照待测物的提取回收率。
6. 若无数据：输出“按验证方案执行，提取回收率无偏差，结果符合要求”。
