### **第一部分：核心准则**
批内/批间准确度及精密度评估必须基于验证方案中定义的接受标准（例如，准确度偏差范围 ≤ ±15.0%，精密度 CV ≤ 15.0%；LLOQ QC 精密度 CV ≤ 20.0%），从质控样品原始测定数据中提取批内/批间统计值（偏差 % 和 CV %）。计算步骤：1) 计算每个浓度水平的准确度偏差 = (测定均值 - 理论值) / 理论值 × 100%；2) 计算精密度 CV = (标准差 / 均值) × 100%；3) 汇总批内范围（每个批次）和批间范围（跨批次均值）；4) 修约至 1 位小数。比较实际范围与接受标准，若所有值符合则判定“符合”；若不符合，报告具体超限值并说明影响。描述时区分 LLOQ QC 与其他浓度水平，确保逻辑闭环：数据提取 → 计算 → 范围汇总 → 标准比较 → 结论陈述。
