# LC-MS/MS法测定食蟹猴EDTA-K2血浆样品中待测物A（游离型）反义链和正义链浓度的生物分析方法学验证SOP

## 目的
规定LC-MS/MS法测定食蟹猴EDTA-K2血浆样品中待测物A（游离型）反义链和正义链浓度的生物分析方法学验证标准操作流程，确保数据质量符合GLP要求。

## 适用范围
适用于LC-MS/MS生物分析方法学验证试验，针对食蟹猴EDTA-K2血浆样品中待测物A（游离型）反义链和正义链的浓度测定验证。

## 操作步骤
1.1 从实验室仪器管理系统查询可用的LC-MS/MS系统编号，确认系统适用性状态

1.2 标准品储备液配制：
    - 从实验室标准品库领取待测物A标准品和内标A标准品
    - 按验证方案92.5.2要求配制储备液
    - 记录配制日期、批号、溶剂种类、体积和浓度

1.3 工作液配制：
    - 按验证方案92.5.3要求从储备液稀释配制工作液
    - 记录稀释倍数、溶剂种类、体积和最终浓度

1.4 标曲/质控样品配制：
    - 按验证方案92.5.4要求配制标准曲线和质控样品（LLOQ、LQC、MQC、HQC）
    - 记录各浓度水平样品数量和配制参数

1.5 样品处理：
    - 按验证方案92.5.5样品处理步骤执行
    - 确保所有样品处理参数一致性

1.6 LC-MS/MS系统运行：
    - 按验证方案92.5.1分析系统要求设置仪器参数
    - 依次运行系统适用性溶液、标准曲线、质控样品、验证样品
    - 每批次不超过验证方案3.13规定的分析批最大进样数

1.7 数据采集：
    - 实时监控仪器运行状态
    - 确保所有色谱图和质谱图完整保存

## 文档记录
记录以下信息至验证试验原始记录表：
- 验证试验编号：SS25071BV02
- LC-MS/MS系统编号和使用日期
- 标准品/内标批号、储备液和工作液配制参数（日期、体积、浓度、稀释倍数）
- 标曲和质控样品配制明细（浓度水平、样品数量）
- 各验证项目样品编号和处理时间
- 系统运行参数设置记录
- 验证负责人和操作人员签名
- 数据采集文件路径和原始数据文件名

## 异常处理
1. 系统适用性不符合接受标准（3.1）：暂停试验，检查仪器状态，重新运行系统适用性溶液
2. 标准曲线不符合接受标准（3.6）：重新配制工作液，按2.1-2.3重新配制标曲
3. 质控样品准确度/精密度不符合接受标准（3.7-3.8）：检查样品处理过程，重新配制质控样品重测
4. 稳定性考察不符合接受标准（3.11.5）：按验证方案规定处理，记录不符合原因和处理措施
5. 仪器故障：立即停止运行，记录故障现象，通知仪器维护人员，报告验证负责人和质量保证单位
6. 所有异常情况须记录详细原因、处理措施和结果，提交质量部门审核
