# 生物样品预处理标准操作程序（SOP）

## 目的
规定生物样品预处理过程的标准操作流程，确保GLP合规性、数据完整性和方法验证要求的可重复性。

## 适用范围
适用于LC-MS/MS生物分析方法验证及样品分析中的血浆、血清、尿液等生物基质的蛋白沉淀预处理。

## 操作步骤

### 1.1 仪器设备准备与验证
1.1.1 从实验室LIMS系统查询可用仪器清单，记录以下设备编号：
    - 冷冻离心机（校准证书有效期内）
    - 超声清洗仪（功率校准合格）
    - 分析天平（校准状态良好）
1.1.2 准备冰浴（0-4°C）、移液器（10-1000μL范围）、耗材（1.5mL离心管、0.22μmPTFE滤膜）
1.1.3 准备冰冷沉淀剂（乙腈/甲醇，-20°C储存），温度验证≤-18°C

### 1.2 样品解冻与预处理
1.2.1 将冻存样品置于4°C冰浴解冻，时间≥30分钟，至完全液化无冰晶
1.2.2 轻柔涡旋混匀30秒，目视确认无明显分层或沉淀
1.2.3 平衡至室温（18-25°C），时间5±1分钟

### 1.3 蛋白沉淀处理
1.3.1 精准移取50±2μL样品至标号1.5mL离心管（天平称量验证）
1.3.2 加入200±5μL冰冷沉淀剂（-20°C），涡旋混匀30±5秒
1.3.3 超声处理5±1分钟，频率40±2kHz，功率控制在设定值±10%
1.3.4 **离心处理**：
    - 温度：4±2°C
    - 转速：14000±500rpm  
    - 时间：15±1分钟
    - RCF：16160×g（参考值）

### 1.4 上清液转移与二次处理
1.4.1 离心完成后静置2±1分钟，避免扰动沉淀
1.4.2 移取200±10μL上清液至新1.5mL离心管（天平验证体积）
1.4.3 加入50±2μL移动相A（0.1%甲酸水溶液），涡旋混匀15±5秒
1.4.4 **滤膜过滤**：
    - 0.22μm PTFE滤膜
    - 离心：12000±500rpm，5±1分钟，4°C

### 1.5 LC-MS/MS进样准备
1.5.1 移取100±5μL滤后上清液至2mL进样瓶
1.5.2 密封进样瓶，激光打印标签（样品编号、处理时间、操作员）
1.5.3 置于4°C自动进样器样品盘，位置按LIMS分配方案

### **记录完整性要求**：
1. **实时记录**：所有实际参数必须操作后5分钟内记录
2. **双人核查**：关键体积转移≥50μL需第二人复核签字
3. **仪器追溯**：设备编号→校准证书→维护记录完整链路
4. **QA审核**：24小时内完成，电子签名符合21 CFR Part 11

## 异常处理

### 2.1 关键设备故障
|异常现象|处理流程|记录要求|
|--------|--------|--------|
|离心机报警/达不到设定转速|停止→更换设备→重新处理→偏差报告|故障代码、实际参数、影响样品数|
|超声仪功率异常|更换超声仪→延长处理2分钟|实际功率读数、设备编号|
|冰浴温度>6°C|更换冰浴→重新解冻|实际温度、延长解冻时间|

### 2.2 样品处理异常
|异常现象|判定标准|纠正措施|
|--------|--------|--------|
|解冻后有明显沉淀|沉淀体积>原样品10%|废弃重解冻或报告QA|
|上清液体积<180μL|天平称量确认|重新制备并记录损失原因|
|滤后液体浑浊|目视确认或0.45μm重滤|记录浑浊程度，判定是否可用|

### 2.3 **标准化偏差处理流程**：
```
1. 立即暂停后续操作 → 隔离受影响样品
2. 拍照存档（异常现象+时间戳）
3. 填写《实验室偏差记录表》
4. 24小时内报告QA + 直接主管
5. 执行CAPA（纠正预防措施）
6. 偏差关闭需QA签字确认
```
