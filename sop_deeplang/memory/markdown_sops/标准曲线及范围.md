## 一、核心填写规则

1. **数据来源与统计参数定义**：从验证方案（VP）中提取标准曲线回归方程、标准样品浓度点数据（如表7和表8所示）。计算线性范围时，使用最小浓度点（LLOQ）和最大浓度点（ULOQ）作为边界，修约为三位小数（例如：0.100 ng/mL）。回归相关系数（R²）阈值不低于0.99，确保线性相关性。无折算因子适用时，直接使用原始浓度单位（ng/mL）；若VP指定折算（如游离型/盐型），则在计算前应用公式：实际浓度 = 原始浓度 × {{Conversion_Factor}}，修约为三位小数。

2. **合规性判断**：比较计算得到的线性范围与VP中预设范围，确保覆盖预期区间。若数据符合，判定为“可达”；若不符合，添加分支描述：“结果表明，本方法线性范围为{{Calculated_Range}}，未达预期间隔{{Expected_Range}}，需进一步优化。” 典型曲线图（图2）必须基于实际数据生成，标注回归方程和R²值。

3. **修约与精度要求**：所有浓度值修约至三位小数，回归系数保留三位小数。线性范围表述使用“可达{{LLOQ}}~{{ULOQ}} ng/mL”，避免四舍五入误差导致的偏差。

4. **无数据处理**：若VP中无具体数据，仅有方案声明，则直接表述“标准曲线及范围按验证方案测定”，并引用预设表/图占位符。确保描述逻辑闭环：数据提取 → 范围计算 → 结果判定。
