# 生物样品分析前处理标准操作程序（SOP）

## 目的
规定生物样品接收、解冻、预处理及仪器分析准备的标准操作流程，确保实验数据符合GLP 21 CFR Part 58和21 CFR Part 11电子记录要求。

## 适用范围
适用于LC-MS/MS、GC-MS等生物分析方法的生物基质样品预处理，包括血浆、血清、尿液、组织匀浆、全血等生物样品。

## 操作步骤

**1. 样品接收与标识**  
1.1 从21 CFR Part 11合规的LIMS系统接收实验样品，核对链 custody记录  
1.2 验证样品编号、接收日期、接收温度（-80°C/-20°C/冰冻/常温）  
1.3 记录样品接收状态，立即置于指定温度储存  

**2. 样品解冻**  
2.1 从冷冻库/冰箱取出样品，置于冰浴（0-4°C）中缓慢解冻  
2.2 解冻时间≤2小时，每30分钟轻柔摇晃1次促进均匀解冻  
2.3 解冻完成后，擦干管外冷凝水，立即称量样品重量（精度0.1mg）  

**3. 样品预处理**  
3.1 从仪器管理系统查询可用离心机编号，确认校准状态  
3.2 准确移取[验证方案指定体积] μL基质样品至15mL聚丙烯离心管  
3.3 加入[验证方案指定体积] μL内标工作液，涡旋混合30秒  
3.4 加入[验证方案指定体积] mL蛋白沉淀剂，涡旋混合60秒  
3.5 离心处理：4°C，12,000×g，10分钟  
3.6 转移[验证方案指定体积] μL上清液至新离心管  
3.7 氮气吹干（40°C，水浴加热），吹干时间≤30分钟  
3.8 加入[验证方案指定体积] μL复溶液，涡旋混合30秒，超声1分钟  

**4. 分析准备与质控**  
4.1 转移[验证方案指定体积] μL处理后样品至进样盘  
4.2 按验证方案设置色谱条件（流速、柱温、进样体积、梯度程序）  
4.3 设置质谱参数（离子源、MRM转换、碰撞能量）  
4.4 运行分析序列：空白→质控样（LLOQ、LQC、MQC、HQC）→分析样品

**电子记录要求（21 CFR Part 11）：**
- 使用验证合格的LIMS/LES系统
- 自动捕获时间戳、用户ID、仪器ID
- 保留原始数据、审计追踪、电子签名

## 异常处理

**1. 解冻异常（>2小时或反复冻融）**
```
- 标记"解冻偏差"，排除定量分析
- 报告QA部门，填写偏差报告(DRF-XXX)
```

**2. 离心设备故障**
```
- 暂停操作，使用备用验证合格离心机
- 记录：故障时间+备用设备编号+QA通知时间
```

**3. 样品体积不足**
```
- 记录实际体积，计算体积校正因子
- 获得研究主任电子批准后继续
```

**4. 氮吹不完全**
```
- 延长吹干时间至45分钟或更换氮气源
- 记录延长原因和最终状态
```

**5. 仪器报警/序列中断**
```
- 暂停，重新注入LQC验证系统状态
- 记录报警代码、恢复时间、质控结果
```

**偏差报告流程：**
```
所有偏差→24小时内电子报告QA→评估影响→决定是否影响数据完整性
```

**版本控制：** 本SOP为通用的生物样品前处理框架，具体体积参数（X、Y、Z、V）及仪器方法按项目验证方案补充。
