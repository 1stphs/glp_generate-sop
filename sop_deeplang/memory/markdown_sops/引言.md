# LC-MS/MS方法学验证样品处理SOP

## 目的
规定LC-MS/MS方法学验证中样品处理的标准操作流程，确保验证结果符合GLP要求。

## 适用范围
适用于LC-MS/MS方法学验证中的标准曲线、质控样品及全血样品处理。

## 操作步骤
1.1 从实验室仪器管理系统查询可用的分析系统编号

1.2 标准品储备液配制：
    - 按照验证方案2.5.2中规定的比例配制标准品储备液
    - 记录配制日期、时间和溶剂批号

1.3 内标储备液配制：
    - 按照验证方案2.5.2中规定的比例配制内标储备液
    - 记录配制日期、时间和溶剂批号

1.4 工作液配制：
    - 按照验证方案2.5.3中规定的稀释比例配制工作液
    - 记录各浓度工作液的配制时间和体积

1.5 标准曲线/质控样品配制：
    - 按照验证方案2.5.4中规定，在空白基质中加入相应工作液配制
    - 充分混匀，确保均匀性
    - 记录配制浓度、基质类型和配制时间

1.6 样品处理步骤：
    - 按照验证方案2.5.5中规定的样品处理流程执行
    - 记录每一步的具体操作参数和处理时间

1.7 LC-MS/MS分析：
    - 使用验证方案2.5.1中规定的分析系统参数
    - 按顺序进样系统适用性样品、标准曲线、质控样品和验证样品

## 文档记录
记录以下信息至方法学验证记录表：
| 日期 | 时间 | 操作步骤 | 仪器编号 | 样品编号 | 浓度 | 基质类型 | 操作人员 | 备注 |
|------|------|----------|----------|----------|------|----------|----------|------|
|      |      |          |          |          |      |          |          |      |

- 所有标准品、内标批号和有效期
- 工作液配制记录（浓度、日期、溶剂）
- 样品处理参数记录
- LC-MS/MS仪器运行参数
- 操作人员签名和审核签名

## 异常处理
1. 如发现标准品或内标不足，按验证方案2.4规定处理剩余标准品返还
2. 如分析系统异常，立即停止进样，记录异常现象，报告验证负责人
3. 如样品处理步骤出现偏差，暂停操作，记录详细偏差情况，通知QA检查员
4. 任何偏离按GLP报告"偏离"章节规定的流程处理
