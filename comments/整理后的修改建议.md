# 毒蛇报告 - 完整修改建议（优先级排序）

## 🔴 紧急修复（立即执行，否则系统不可用）

### 1. JSON解析方式 - 彻底废弃正则表达式
**问题**：`extract_json_from_text` 使用 `re.search(r'```json...')` 石器时代解析方式
**修复方案**：
- 使用 OpenAI/Claude 的 `response_format={"type": "json_schema"}` + Structured Outputs
- 集成 Pydantic 模型做强类型校验
- 完全移除所有正则表达式解析逻辑

**代码示例**：
```python
from pydantic import BaseModel

class SOPOutput(BaseModel):
    content: str
    score: float
    issues: list[str]

# OpenAI 调用
response = client.chat.completions.create(
    model="gpt-4o",
    response_format={"type": "json_schema", "json_schema": SOPOutput.model_json_schema()},
    messages=[...]
)
```

---

### 2. 存储方案 - 废弃文件系统伪数据库
**问题**：`memory/rules/` 使用裸 JSON + `open(file, "w")` 导致 race condition
**修复方案**：
- **短期**：PostgreSQL + 事务锁（防并发冲突）
- **长期**：Pinecone/Chroma 向量数据库（语义检索 + 版本控制）
- 集成 LangGraph persistence 做状态持久化

**迁移步骤**：
1. 创建 PostgreSQL 表结构（rules, memory, versions）
2. 添加行级锁：`SELECT ... FOR UPDATE`
3. 所有文件读写改为数据库事务

---

## 🟠 核心架构重构（1周内完成，否则成本失控）

### 3. Agent 数量精简 - 从5个砍到3个
**问题**：5-agent 飞轮（Writer → Simulator → Reviewer → Reflector → Curator）串行阻塞，25次 API 调用
**修复方案**：
- **合并**：Simulator + Reviewer + Reflector → 超级 Reviewer Agent
- **保留**：Writer（生成）+ 超级 Reviewer（评估）+ Curator（沉淀）
- **并行化**：Writer 和 Simulator 可并行执行

**效果**：
- API 调用从 25 次降至 8-10 次
- 成本节省 70%
- 延迟从"泡咖啡时间"降至秒级

---

### 4. 多模型混合策略 - 打破"互相吹捧"
**问题**：全用 Claude 3.5 Sonnet 导致 AI 集体自嗨
**修复方案**：
```python
# Writer（生成）：便宜快速模型
writer_agent = create_agent(model="gpt-4o-mini")  # 或 claude-haiku

# 超级 Reviewer（评估）：毒舌模型
reviewer_agent = create_agent(
    model="grok-4-1-fast-reasoning",
    system_prompt="你是最刻薄的 FDA 审计员，必须挑刺..."
)

# Curator（沉淀）：深度推理模型
curator_agent = create_agent(model="claude-opus")  # 或 grok-2
```

**效果**：
- 不同模型"审美"差异消除互相吹捧
- 小模型占 70% 调用，成本反降
- GLP 合规质量从"看起来像"变成"真能过 FDA"

### 6. 异步并行 - 解决串行阻塞
**问题**：5 个 Agent 串行等待，延迟累加
**修复方案**：
```python
import asyncio

async def parallel_generation():
    tasks = [
        writer_agent.run_async(chapter_1),
        writer_agent.run_async(chapter_2),
        simulator_agent.run_async(context)  # 可与 Writer 并行
    ]
    results = await asyncio.gather(*tasks)
```

**配合 LangGraph**：
- 使用 checkpointing 做状态恢复
- ThreadPoolExecutor 用于 Agent 内部并行

### 7. Human-in-the-Loop - GLP 合规必须
