# 小分子模板规则

## GLP 报告生成规则

### 必需章节
- **16.2 声明**：必须包含 GLP 合规声明
- **样品信息**：编号、名称、收样日期
- **分析结果**：含量、纯度、水分等关键指标
- **质量控制**：空白对照、标准品、平行样

### 数据格式要求
- 含量和纯度使用百分比（%）
- 水分含量保留一位小数
- 日期格式：YYYY-MM-DD

### 质量控制标准
- 所有对照样必须标注"合格"或"不合格"
- 平行样 RSD < 3%

## 方法验证规则

### 验证参数
- 准确度：±5% 范围内
- 精密度：RSD < 3%
- 线性范围：需明确上下限

### 步骤编写规范
- 每个步骤必须可独立执行
- 包含具体的操作参数（温度、时间、体积等）
- 明确质量控制点
